キーボードショートカット

この一覧は実装されているキー操作のみを記載しています。

0. 使い方の前提

  • 同じキーでも、フォーカス位置（プレビュー / ファイル名入力欄 / 画面全体）で動作が異なります
  • 優先順位は、一般的に「入力欄のイベント」→「プレビューのイベント」→「画面全体のイベント」です
  • 分割処理実行中は、ほとんどの操作が無効化されます

1. プレビュー上で使えるキー

キー 動作
PageUp 1ページ戻る
PageDown 1ページ進む
Ctrl + PageUp 10ページ戻る
Ctrl + PageDown 10ページ進む
Home 先頭ページへ移動
End 最終ページへ移動
Enter 現在ページに分割点を追加
Delete 現在ページの分割点を削除
Shift + Enter 分割を実行
z ズームイン
Z (Shift + z) ズームアウト
d ズームを100%に戻す

補足:

  • Enter は「分割点を追加」、Shift + Enter は「分割実行」です
  • ズーム操作はプレビュー領域にフォーカスがあるときのみ有効です

2. ファイル名入力欄で使えるキー

キー 動作
Tab ファイル名を保存して次セクションへ移動
Enter ファイル名を保存して次セクションへ移動
Shift + Enter 分割を実行
Delete 現在セクションの開始分割点を削除

※ 入力欄からフォーカスが外れたとき（FocusOut）も、ファイル名は保存されます。

補足:

  • Enter と Tab はどちらも「保存して次へ」の同じ動作です
  • 連続入力では Enter 連打でセクションを順に進められます

3. グローバルキー（画面全体）

キー 動作
Shift + Enter 分割を実行
Ctrl + ↑ 前のセクションへ移動
Ctrl + ↓ 次のセクションへ移動

補足:

  • グローバルキーは bind_all で登録されています
  • 入力欄やプレビュー側で同じキーが処理される場合は、そちらが優先されることがあります

4. 補足

  • テンキーの Enter にも対応しています
  • Z は Shift + z と同義です
  • 誤操作を防ぐには、実行前にプレビュー側へ一度クリックしてフォーカスを明示してください

5. 実務でよく使う組み合わせ

  • ページ確認しながら分割点を置く
    • PageDown → Enter を繰り返す
  • セクション名を素早く付ける
    • 入力 → Enter（次セクションへ）
  • 最終確認後に実行
    • Ctrl + ↑/↓ でセクション移動 → Shift + Enter
