PDF Toolbox 取扱説明書

この文書は、PDF Toolbox の基本操作を利用者向けにまとめたものです。 ホーム画面から各機能へ移動し、入力の選び方、保存先の指定、実行中の制約を確認できます。

0. はじめに

0.1 想定する利用シーン

  • 1 つの PDF を章ごとに分けたい
  • 複数 PDF を順番どおりにまとめたい
  • 複数 PDF から必要ページだけを抜き出して 1 つにしたい
  • PDF や ZIP をまとめて圧縮したい
  • 注釈やフォームを編集不可の配布用 PDF にしたい
  • PDF を全ページ JPEG にしたい

0.2 この文書で分かること

  • 起動からホーム画面までの流れ
  • 6 つの機能の基本操作
  • 出力ルールと主な制約
  • 実行中にできない操作
  • よくあるトラブルと対処

0.3 用語の整理

  • PDF 分割: 1 つの PDF を複数 PDF に分ける機能
  • PDF 抽出: 複数 PDF から任意ページを集めて 1 つの PDF を作る機能

1. 起動方法

1.1 動作条件

  • 対応 OS: Windows を前提に確認しています
  • ソース実行時の Python 設定: pyproject.toml では >=3.14
  • 主な入力形式: .pdf
  • 圧縮機能では .zip とフォルダ入力にも対応します

1.2 ソース実行

  1. Python を準備します。
  2. 仮想環境を有効化します。
  3. 依存関係をインストールします。
pip install -U pyside6 pillow pymupdf pikepdf
  1. アプリを起動します。
python main.py
  1. 起動時にはスプラッシュ画面が表示され、その後ホーム画面へ切り替わります。

1.3 実行ファイル配布版

  • 配布された .exe をダブルクリックして起動します
  • 初回起動時は SmartScreen 警告が出る場合があります
  • 起動後はホーム画面から使いたい機能カードを選びます

2. ホーム画面の見方

ホーム画面には次の 6 つの機能カードがあります。

  • PDF 分割
  • PDF 結合
  • PDF 抽出
  • PDF 圧縮
  • PDF → JPEG
  • PDFフラット化

カードを押すと対応画面へ移動します。 各画面の作業状態は保持されるため、未実行のままホームへ戻ると確認ダイアログが出る場合があります。

3. PDF 分割

3.1 どんなときに使うか

  • 1 冊の PDF を章単位で分けたいとき
  • ページ境界ごとに別ファイルとして保存したいとき

3.2 基本操作

  1. ホーム画面で「PDF 分割」を押します。
  2. 分割したい PDF を開きます。
  3. プレビューで目的ページへ移動します。
  4. Enter または画面操作で分割点を追加します。
  5. 必要に応じて右側のセクション一覧でファイル名を編集します。
  6. 保存先フォルダを選びます。
  7. 分割を実行します。

3.3 主な仕様

  • 分割点はページ境界ごとに設定します
  • 分割点のない状態では 1 つの出力になります
  • セクションごとに出力ファイル名を変更できます
  • 不正な文字や Windows の予約名は自動的に安全な名前へ補正されます
  • プレビューは 0.5 倍から 3.0 倍まで調整できます

3.4 実行中の注意

  • 分割処理の実行中は PDF を開き直せません
  • 実行中はホームへ戻れません
  • 終了しようとすると確認ダイアログが表示されます

4. PDF 結合

4.1 どんなときに使うか

  • 複数の PDF を 1 つにまとめたいとき
  • 結合前に順番を入れ替えたいとき

4.2 基本操作

  1. ホーム画面で「PDF 結合」を押します。
  2. 結合したい PDF を複数追加します。
  3. 一覧またはドラッグ＆ドロップで順番を調整します。
  4. 必要に応じて不要な入力を削除します。
  5. 保存先 PDF を選びます。
  6. 結合を実行します。

4.3 主な仕様

  • 入力は PDF ファイルのみです
  • 重複したパスは自動的に無視されます
  • 選択中の入力を上下移動できます
  • 保存先ファイルが既に存在する場合は上書き確認が表示されます

4.4 実行中の注意

  • 実行中は入力一覧、並び順、保存先を変更できません
  • 実行中はホームへ戻れません
  • 終了時には中断確認が表示されます

5. PDF 抽出

5.1 どんなときに使うか

  • 複数の PDF から必要ページだけを 1 つにまとめたいとき
  • 元ファイルの順番と関係なく、配布用 PDF を組み直したいとき

5.2 画面の考え方

  • Source: 元になる PDF と各ページの一覧です
  • Target: 抽出後の PDF に含めるページの並びです

Source から Target へページを集め、最後に 1 つの PDF として保存します。

5.3 基本操作

  1. ホーム画面で「PDF 抽出」を押します。
  2. 抽出元の PDF を 1 つ以上追加します。
  3. Source 側でページを選択します。
  4. 次のいずれかで Target に追加します。
  • ボタン操作で追加する
  • ページをダブルクリックする
  • Source から Target へドラッグ＆ドロップする
  1. Target 側で順番を並び替えます。
  2. 保存先 PDF を選びます。
  3. 抽出を実行します。

5.4 選択のしかた

  • 通常クリック: 単一選択
  • Ctrl + クリック: 選択の追加・解除
  • Shift + クリック: 同じ PDF 内で範囲選択

5.5 主な仕様

  • Source には複数 PDF を追加できます
  • Target には同じページを重複して追加できます
  • Target の順番がそのまま出力 PDF のページ順になります
  • 保存先ファイルが既に存在する場合は上書き確認が表示されます
  • Source と Target でサムネイル表示倍率を個別に調整できます

5.6 実行中の注意

  • 実行中は Source、Target、保存先を変更できません
  • 実行中はホームへ戻れません
  • 終了時には中断確認が表示されます

6. PDF 圧縮

6.1 どんなときに使うか

  • PDF をまとめて軽くしたいとき
  • フォルダ単位または ZIP 単位で一括処理したいとき

6.2 基本操作

  1. ホーム画面で「PDF 圧縮」を押します。
  2. 入力として PDF、フォルダ、ZIP のいずれかを追加します。
  3. 保存先フォルダを選びます。
  4. 圧縮モードを選びます。
  • 非可逆圧縮
  • 可逆圧縮
  • 両方
  1. 必要に応じて JPEG 品質、PNG 品質、DPI、可逆圧縮オプションを調整します。
  2. 圧縮を実行します。

6.3 主な仕様

  • 入力として PDF、ZIP、フォルダを受け付けます
  • PDF / ZIP / フォルダ以外は追加されません
  • ZIP の再帰探索深度は制限されています
  • 非可逆圧縮では、PDF に埋め込まれた透過付き PNG や soft mask 付き画像は、透過を保持したまま PNG として再圧縮します
  • 非可逆圧縮では、PDF に埋め込まれた CMYK JPEG 画像も RGB 系へ正規化してから再圧縮します
  • PNG 量子化では、pngquant が利用可能ならそれを優先し、利用できない場合や実行失敗時は Pillow フォールバックで処理を継続します
  • 完了時に成功件数、失敗件数、スキップ件数、圧縮率が表示されます

6.4 実行中の注意

  • 実行中は入力一覧や保存先を変更できません
  • 実行中はホームへ戻れません
  • 終了時には中断確認が表示されます

7. PDFフラット化

7.1 どんなときに使うか

  • 注釈を書き込んだ PDF を配布前に固定したいとき
  • 入力フォーム付き PDF を編集不可の完成版にしたいとき
  • フォルダ単位で複数の PDF をまとめて平坦化したいとき

7.2 基本操作

  1. ホーム画面で「PDFフラット化」を押します。
  2. 入力として PDF またはフォルダを追加します。
  3. 必要に応じて不要な入力を削除します。
  4. 「フラット化を実行」を押します。
  5. 上書き確認が出た場合は、続行方法を選びます。
  6. 完了ダイアログで成功、失敗、スキップ件数を確認します。

7.3 出力ルールと上書き確認

  • 出力ファイル名は元 PDF と同じ場所に 元名_flattened.pdf を付けて保存されます
  • 既に同名の _flattened.pdf がある場合は一括上書き確認が表示されます
  • 「はい」を選ぶと衝突分も含めて上書きします
  • 「いいえ」を選ぶと衝突分だけスキップし、他の入力は続行します

7.4 フォルダ入力と実行中の制約

  • フォルダを追加した場合は、その配下を再帰的に探索して .pdf だけを対象にします
  • PDF 以外のファイルや存在しない入力は自動的にスキップされます
  • 実行中は入力の追加、削除、一覧クリア、ホームへの移動はできません
  • 終了しようとすると確認ダイアログが表示され、安全にキャンセルされた後で終了します

7.5 注釈、フォーム、暗号化 PDF、ロック中ファイル

  • 注釈やフォーム field、widget は平坦化後に編集対象として残りません
  • フォーム付き PDF では、見た目を保持したまま通常のページ内容へ焼き込みます
  • 暗号化 PDF は処理できず、失敗件数として集計されます
  • 出力先ファイルが他のアプリで開かれている場合も失敗として扱われます
  • broken appearance 系 PDF は処理継続を優先し、クラッシュよりも failure / skip 集計を優先する設計です

8. PDF → JPEG

7.1 どんなときに使うか

  • PDF をページ画像として配布したいとき
  • 各ページを JPEG にして他ツールへ渡したいとき

7.2 基本操作

  1. ホーム画面で「PDF → JPEG」を押します。
  2. 変換したい PDF を 1 つ選びます。
  3. 先頭ページプレビューを確認します。
  4. JPEG 品質を調整します。
  5. 保存先フォルダを選びます。
  6. 「JPEG 書き出しを実行」を押します。
  7. 完了後に出力先サブフォルダを確認します。

7.3 出力ルール

  • 入力は単一 PDF のみです
  • 対象ページは全ページ固定です
  • 出力ファイル名は PDF名_001.jpg 形式です
  • 保存先は 保存先/PDF名/ サブフォルダです
  • 同名 JPEG がある場合は実行前に一括上書き確認が表示されます
  • 透明要素を含むページは白背景へ合成して保存されます

7.4 実行中の注意

  • 実行中は入力 PDF、保存先、品質を変更できません
  • 実行中はホームへ戻れません
  • 終了時には中断確認が表示されます

7.5 現時点で未対応の項目

  • ページ範囲指定
  • PNG 出力
  • 命名テンプレート変更
  • DPI 指定

9. トラブルシューティング

症状 主な原因 対処
実行ボタンが押せない 入力または保存先が不足している 画面上の必須項目を設定する
一部の入力が追加されない 非対応形式、存在しないファイル、重複入力 対応形式か確認し、同じパスを重ねて追加しない
上書き確認が出る 保存先に同名ファイルがある 上書きするか、別の保存先を選ぶ
実行中にホームへ戻れない 実行中ジョブ保護のため制限されている 完了を待つ
暗号化 PDF が処理できない パスワード保護されている 復号後の PDF を入力する
保存時に他アプリ利用中と出る 出力先ファイルが開かれている PDF ビューアなどを閉じてから再実行する
JPEG の背景が白になる 透明要素を白背景へ合成する仕様 異常ではなく仕様どおり
