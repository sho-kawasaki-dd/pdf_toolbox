PDF分割アプリ 取扱説明書（非開発者向け）

この文書は、PDFを分割して保存するための操作手順を説明します。

0. はじめに

想定する利用シーン

  • 1つのPDFを章ごとに分割したい
  • ページ単位で個別PDFにしたい
  • セクションごとに分かりやすいファイル名を付けたい

この文書で分かること

  • 起動から分割完了までの基本手順
  • 効率的な操作方法（マウス＋キーボード）
  • エラーが出たときの対処

1. このアプリでできること

  • PDFを開いてページを確認できます
  • 好きな位置に分割点を設定できます
  • セクション（分割後ファイル）ごとにファイル名を指定できます
  • 分割を実行して、複数のPDFとして保存できます

2. 起動方法

2.0 動作条件

  • 対応OS: Windows（本ワークスペースの検証環境）
  • Python実行時: pyproject.toml 設定は >=3.14
  • 入力形式: .pdf

2.1 ソース実行（開発環境がある場合）

  1. Pythonを準備します（pyproject.toml 設定: >=3.14）

  2. （推奨）仮想環境を有効化します

  3. 依存関係をインストールします

    pip install -U customtkinter pillow pymupdf

  4. アプリを起動します

    python main.py

  5. 起動時表示を確認します

起動直後に、中央へ白背景のスプラッシュ（アプリアイコン）が表示されます。
スプラッシュは最小1秒表示され、初期化完了後に自動で閉じます。

2.2 実行ファイル配布版（.exe など）

  • 配布された実行ファイルをダブルクリックして起動します
  • 初回起動時はSmartScreen警告が出る可能性がありますが、「詳細情報」→「実行」を選択してください
  • 起動直後に、中央へ白背景のスプラッシュ（アプリアイコン）が表示されます
  • 操作方法はこの文書と同じです

3. 画面の見方

  • 左エリア
    • PDFプレビュー
    • 分割バー（色付きの帯）
    • ページ移動ボタン
    • 分割点追加/削除ボタン
  • 右エリア
    • 「PDFを開く」
    • 「すべての分割点をリセット」
    • 「全体を1ページずつ分割する」
    • セクション情報・ファイル名入力
    • 「分割を実行」

3.1 分割バーの見方

  • 色の帯: セクションの区切りを表します
  • 黒い縦線: 分割点です
  • 白い枠: 現在アクティブなセクションです
  • 白い逆三角: 現在表示中のページです

分割バーはクリック・ドラッグでページ移動できます。

4. 基本操作（通常フロー）

  1. 「PDFを開く」を押して対象PDFを選びます
  2. 分割したいページへ移動します
  3. 「現在のページに分割点を設定する」で分割点を追加します
  4. 必要に応じてセクションごとの出力ファイル名を入力します
  5. 「分割を実行」を押し、保存先フォルダを選びます
  6. 完了メッセージが表示されたら保存完了です

4.1 おすすめの実務フロー

  1. 大まかに分割点を追加
  2. 「前のセクション」「次のセクション」で境界を確認
  3. 必要なセクションだけファイル名を編集
  4. 最後にまとめて「分割を実行」

4.2 1ページずつ分割する場合

  1. 「全体を1ページずつ分割する」を押す
  2. 必要なら一部セクションのファイル名だけ変更
  3. 「分割を実行」で保存

注意: この操作は既存の分割点を上書きします。

5. ファイル名のルール

入力したファイル名は自動で整形されます。

  • 不可文字 \\ / : * ? " < > | は _ に置換されます
  • 前後の空白や記号は整理されます
  • 拡張子 .pdf は自動調整されます
  • 無効な名前（空、予約語など）は既定名に戻ります

5.1 具体例

  • 入力: ” 請求書:2026/03 ” → 出力: 請求書_2026_03.pdf
  • 入力: CON（予約語）→ 出力: output_partN.pdf（既定名）
  • 入力: 報告書.pdf → 出力: 報告書.pdf（維持）

既定名

  • output_part1.pdf, output_part2.pdf …

同名ファイルが既にある場合

既存ファイルと重複しないように、自動で連番が付きます。

  • 例: output_part1.pdf → output_part1 (1).pdf

6. よく使う操作

  • 現在ページの分割点を消す: 「現在のページの分割点を消去」
  • すべての分割点を消す: 「すべての分割点をリセット」
  • 1ページずつ分割したい: 「全体を1ページずつ分割する」
  • セクション単位で移動: 「前のセクション」「次のセクション」

6.1 入力欄の便利操作

  • Enter / Tab: 名前保存 → 次セクションへ移動
  • Shift + Enter: そのまま分割実行
  • Delete: 現在セクション開始の分割点を削除

7. 分割実行中の注意

  • 実行中は主要操作が無効になります
  • 実行中に終了しようとすると確認ダイアログが表示されます
  • 実行中はPDFを開き直せません

7.1 安全に運用するコツ

  • 出力先フォルダを毎回確認する
  • 同名ファイルが多いフォルダでは、完了後に生成ファイル数を確認する
  • 大きなPDFでは処理時間が長くなるため、完了メッセージが出るまで待つ

8. エラー時の対処

保存エラー（ファイル使用中）

  • メッセージ例: 保存先ファイルが他アプリで使用中
  • 対処:
    1. 該当PDFを開いているアプリ（ビューア等）を閉じる
    2. もう一度分割を実行する

分割対象がない

  • 分割対象セクションが存在しない場合はエラーになります
  • PDFを開き直し、分割点を確認してから再実行してください
